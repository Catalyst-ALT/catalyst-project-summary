Project Description

Catalyst is an app that enhances an artist's creative process through the power of AI

Catalyst is an innovative app that empowers artists by integrating openAI's API to generate unique and creative prompts, fostering inspiration and motivation for artistic creation. Our mission is to preserve the magic of artistic generation by placing it in the hands of creators, ensuring that technology enhances rather than overshadows the creative process. The backend developed a RESTful API and used user interactions to parameterize prompts sent to openAI's API. Users make selections tailored to different artistic mediums, yielding over 7000 personalized prompt combinations. After hitting the generate button, the backend processes user choices, queries the AI model, and refines prompts to maintain accuracy. The temperature parameter controls creativity, ensuring desired results. The Create page, built using React, offers dynamic tools like Stopwatch and Pomodoro Timers, along with a Notebook for jotting down insights. This curated workspace enhances an artist's studio experience, centralizing tools for diverse creative processes. The Lexicon feature redefines traditional dictionaries. Users can search prompt words for meanings, examples, jokes, and even associated colors. Unlike the Generate feature, Lexicon enables input autonomy and contextualizes prompts. The app's aesthetic, designed with care and minimalism, offers a blank canvas for imagination. Tailwind CSS provides flexibility for responsive design and fluid user experience. UI considerations utilize React for step-by-step parameter selection, enhancing user flow and instruction. A redesigned nav bar grants autonomy in navigating between parameter categories. Muse, our custom loading animation, adds charm to the app with captivating animations. Catalyst's essence lies in bridging artistic innovation and AI technology, enabling creators to break through creative barriers while maintaining the vital human touch. Our app embraces the harmony between technological advancement and individual artistic expression, providing the perfect balance of innovation and creativity.
